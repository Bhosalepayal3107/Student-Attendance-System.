# Student Attendance System

This is a mini project made using Python and Tkinter.

## Features
- Add student
- Mark attendance
- View attendance percentage
- Store data using JSON file

## Technologies Used
- Python
- Tkinter
- JSON

## How to Run
1. Install Python
2. Download the project
3. Run the file:
   python attendance_gui.py
